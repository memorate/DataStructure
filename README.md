### 一、数组 | Array
1) 定义: 数组是n个**相同数据类型**的元素构成的占用一块**连续的内存单元**的有限序列  
2) 数组又分为**有序数组**和**无序数组**，有序数组会对插入的元素进行排序  
#### 1.无序数组
1) 优点: 插入快  
2) 缺点: 查找慢、删除慢、大小固定、只能存储单一元素  
3) 时间复杂度: 查找 -- O(N)、插入 -- O(1)、删除 -- O(N)
#### 2.有序数组
1) 优点: 比无序数组查询快  
2) 缺点: 插入慢、删除慢、大小固定、只能存储单一元素  
3) 时间复杂度: 查找 -- O(logN)、插入 -- O(N)、删除 -- O(N)
